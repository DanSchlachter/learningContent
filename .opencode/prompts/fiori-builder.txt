## Required Annotations per Entity

### List Report Page
```cds
annotate Service.Entity with @(
  UI.HeaderInfo: {
    TypeName: 'Label', TypeNamePlural: 'Labels',
    Title: { Value: titleField },
    Description: { Value: descField }
  },
  UI.LineItem: [
    { Value: field1, Label: 'Col 1' },
    { Value: field2, Label: 'Col 2' },
    { $Type: 'UI.DataFieldForAction', Action: 'Service.Entity/actionName', Label: 'Do It' }
  ],
  UI.SelectionFields: [ filterField1, filterField2 ]
);
```

### Object Page
```cds
annotate Service.Entity with @(
  UI.Facets: [
    { $Type: 'UI.ReferenceFacet', Label: 'Section', Target: '@UI.FieldGroup#Key' },
    { $Type: 'UI.ReferenceFacet', Label: 'Child Items', Target: 'items/@UI.LineItem' }
  ],
  UI.FieldGroup#Key: {
    Label: 'Section Title',
    Data: [
      { Value: field1 },
      { Value: field2 }
    ]
  }
);
```

### ValueList for Associations and Enums
```cds
annotate Service.Entity:associationField with @(
  Common.ValueList: {
    CollectionPath: 'RelatedEntities',
    Parameters: [
      { $Type: 'Common.ValueListParameterOut',         LocalDataProperty: assoc_ID, ValueListProperty: 'ID' },
      { $Type: 'Common.ValueListParameterDisplayOnly', ValueListProperty: 'name' }
    ]
  },
  Common.Text: associationField.name,
  Common.TextArrangement: #TextOnly
);
```

### Field-level annotations
```cds
annotate Service.Entity with {
  summary       @UI.MultiLineText;
  url           @title: 'URL';
  isDeleted     @UI.Hidden;
  thumbnail_url @title: 'Thumbnail URL';
}
```

## Browse App Specifics
- Use `UI.PresentationVariant` to sort by `created_at DESC` by default
- Use `UI.DataPoint` for content type badge with `Criticality: 3`
- Do NOT show edit actions (read-only entities)
- Show prominent external URL link field

## Admin App Specifics
- Show action buttons in `UI.LineItem` using `UI.DataFieldForAction`
- Status field: use `Common.ValueList` with fixed values (draft, published, archived)
- Category field: ValueList pointing to Categories entity
- Tag fields: ValueList pointing to Tags entity
